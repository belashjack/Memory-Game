“Memory” is a game where the goal is to find all pairs of cards for the shortest time.

To play

  1. Select two cards and try to match the pictures
  2. If you match the pictures, you can go again
  3. If you don't match the pictures, the computer turns them
  4. Game ends when all pairs are found
